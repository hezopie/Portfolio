Registration SiteS

Opportunity

Due to the volume of programs the digital team supported, we needed to be able to produce styling for client-branded registration sites quickly.

Setup

Using a generic style master as a starting point (HTML and SCSS, compiled to CSS), we would run a Gulp task to automatically generate a job folder with the program job number assigned, pull in appropriate client styling and create a simple program-level SCSS/CSS file. Using the program-level file and an approved UI mockup as a reference, we could quickly write any changes for the particular program that differed from the client master.

My Role

  • Project management
  • Task assignment
  • Styling as needed
  • QC and communication with the requestor.

Outcome

Fully styled program sites that could typically be turned around a day or two.
